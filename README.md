1.Reverse the function like this:
```php
<?php

function dec($data, $key) {
	$data = base64_decode($data);
    $result = '';
    $keyLength = strlen($key);
    $dataLength = strlen($data);

    for ($i = 0; $i < $dataLength; $i++) {
        $char = $data[$i];
        $keyChar = $key[$i % $keyLength];
        $decodeChar = chr(ord($char) - ord($keyChar));
        $result .= $decodeChar;
    }

    return $result;
}
```
